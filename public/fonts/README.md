# Fonts Directory

## PDMS Saleem Quran Font

Please download the PDMS Saleem Quran Font TTF file and place it in this directory.

**Download Link:** https://urdunigaar.com/download/pdms-saleem-quran-font-ttf-file-download/

**File Name:** `PDMS_Saleem_QuranFont.ttf` (or rename it to this)

After downloading, place the TTF file here:
`public/fonts/PDMS_Saleem_QuranFont.ttf`

The font will be automatically loaded in the application for displaying Quranic verses.
